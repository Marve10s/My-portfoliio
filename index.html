Featured project

Forkify

The Forkify web application implements the Model-View-Controller (MVC) architecture for convenient and efficient management of the program's logic and presentation. The Model contains data about recipes and prepared dishes. The View is responsible for displaying this data and interacting with the user through the interface. The Controller coordinates the interaction between the Model and the View, handling input data and performing necessary actions. Additionally, the Forkify website is implemented using an object-oriented approach (OOP) in JavaScript to enhance code readability, support, and scalability. This allows for easy management of complex object relationships and ensures modularity and code reusability. The Forkify web application features functions that allow users to search, view, save recipes, and create their own recipes. Interactivity with the user is provided through interaction with the interface and the ability to edit recipes.
